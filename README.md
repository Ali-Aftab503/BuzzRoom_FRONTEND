# Buzz Room - Real-time Messaging App

Buzz Room is a modern, real-time messaging application designed for seamless communication. It features real-time chat rooms, direct messaging (coming soon), and a sleek, user-friendly interface.

## Live Demo

Check out the live application here: [Live Demo Link](https://buzz-room-frontend.vercel.app)

## Features

- **Real-time Messaging**: Instant message delivery using Socket.io.
- **Chat Rooms**: Create and join public chat rooms.
- **Favorites**: Mark rooms as favorites for quick access.
- **User Presence**: See who's online in real-time.
- **Modern UI**: A dark-themed, responsive interface built with Tailwind CSS.
- **Authentication**: Secure login and registration system.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Node.js, Express, Mongoose
- **Real-time**: Socket.io, Web RTC
- **State Management**: React Context API

## Installationm

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd BuzzRoom_FRONTEND
    ```

2.  **Install dependencies:**
    Navigate to the frontend directory:
    ```bash
    cd ChatApp/FRONTEND
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in browser:**
    Visit `http://localhost:5173` (or the port shown in your terminal).

## Usage

1.  **Register**: Create a new account with a username and email.
2.  **Login**: Sign in to your account.
3.  **Create a Room**: Click "New Room" to create a public chat space.
4.  **Join a Room**: Click on any room card to join the conversation.
5.  **Favorite**: Click the bookmark icon to save rooms to your favorites list.
